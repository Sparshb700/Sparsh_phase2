## 3. Welcome to the AttackBox

## 4. The CMD Conundrum
Used `cmd` commands to read a hidden file
Used `dir /a` to list hidden files

## 5. Linux Lore
Used `linux shell` to read a hidden file
Used `ls -la` to locate

## 6. The leak in the list

## 7. WiFi Woes in Wareville
Had to login into the router to change its password to something more strong than `admin`
This is actually very important as people forget to change the password from `admin`

## 8. The App Trap
Found the app which had the permission to access password vault and revoked its access
